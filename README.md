# 🎮 Fundamentos de C# 

## 🛠️ Tecnologias e Estruturas Utilizadas
* **C# / .NET 8**
* Implementação de `Enums` para controle rígido de dados e prevenção de erros de entrada.
* Estruturas de controle de fluxo `Switch/Case` atuando como máquinas de estado.
* Separação de responsabilidades (Lógica separada do Motor de Execução).

## 🚀 O que foi simulado aqui?
Os exercícios fogem do formato convencional de lógica básica e simulam cenários reais da indústria de tecnologia e desenvolvimento de jogos:
1. Triagem de severidade de relatórios de Bugs (QA).
2. Filtros estritos de coleções e Pokédex.
3. Tratamento e logs de erro para suporte técnico.
4. Lógica de Crafting baseada em categorias de itens.
5. Simulação de Localização de UI Multilíngue (LQA).
